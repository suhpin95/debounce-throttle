# debounce-throttle
Basic Implementation of debounce and throttle on Javascript

This repository has been created for reference purpose. The focus here is to understand the basic implementation of debounce and throttle.

### Pre-requisite
1. Closure
2. Arrow function
3. DOM-Events

### Steps to run

1. Clone the repository
2. Check the index.html file
3. Run on the browser
